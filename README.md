# Student Management System (Rust)

A simple, memory-based console application built in Rust to manage student records, scores, and grade calculations.

## Features
- **Add Students**: Create records with unique IDs, names, and ages.
- **Score Management**: Add multiple exam scores to existing students.
- **Reporting**: 
  - View a summary table of all students.
  - View a detailed report for an individual student (Total, Average, and Grade).
- **Automated Grading**: Calculates grades (Excellent, Very Good, etc.) based on average scores.
- **Removal**: Delete student records by ID.
